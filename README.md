# Football Highlight Generator

## 📌 Overview

The project focuses on the automated generation of football match highlights using machine learning models combined with multimodal analysis of match video and commentary audio streams to detect and localize key events such as goals, fouls, corners, and other significant moments, enabling the extraction of concise highlight clips.

## 🎯 Objectives

- Detect important football events from match data
- Combine multimodal inputs (video, audio, text)
- Generate structured highlight outputs

## 🧠 Methodology

The system uses:

- Video processing (frame extraction, temporal modeling)
- Audio analysis (event classification using spectrograms)
- Multimodal fusion techniques for event detection
